# Project Title: BlockRun

BlockRun is a fast-paced 2D platformer focused on precise movement, timing, and momentum. The game emphasizes responsive controls and clean mechanics, challenging players to navigate obstacle-filled levels with increasing difficulty.

## Play the Game
This project is not currently hosted online, but it is fully playable through the downloadable ZIP file. Players can launch the game directly using the included executable without installing any additional software.

## Features
- 2D platformer gameplay with tight, responsive controls
- Movement-focused level design
- Obstacle-based challenges requiring precision and timing
- Clean and minimal visual style
- Progressive difficulty through level design

## How to Play
Use the keyboard to control the player.

- Move: Arrow keys or A/D
- Jump: Spacebar
- Objective: Reach the end of each level while avoiding obstacles

## Installation (Player)
1. Download the ZIP file from the repository
2. Extract the contents
3. Run the `.exe` file to play

## Installation (Developer)
To modify or rebuild the project:
1. Install GameMaker Studio 2
2. Open the `.yyp` project file
3. Run or export the game from the IDE

## Technologies Used
- GameMaker Studio 2
- GML (GameMaker Language)
- Custom movement and collision logic
